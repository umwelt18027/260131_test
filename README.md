# 🐍 Snake Game (HTML / CSS / JavaScript)

웹 브라우저에서 실행되는 간단한 스네이크 게임입니다.  
하나의 HTML 파일 안에 **CSS + JavaScript**가 통합되어 있으며, **키보드 방향키**로 조작합니다.

---

## ✨ 기능
- 🎮 방향키로 뱀 이동  
- 🍎 먹이를 먹으면 길이 증가  
- 💥 벽 또는 몸과 충돌 시 게임 오버  
- 🧮 점수 표시  
- 🔄 방향키 입력으로 게임 재시작  

---

## 📁 파일 구성
- `index.html` : 게임 전체가 포함된 단일 파일

---

## 🚀 실행 방법
1. 저장소를 클론하거나 ZIP 파일로 다운로드한다.  
2. `index.html` 파일을 브라우저로 열면 바로 실행된다.

---

## 🧩 게임 구조 설명

### 🎨 Canvas
- 크기: 400×400  
- 격자 단위: 20px  

### 🐍 Snake 로직
- 배열로 몸통을 관리  
- 머리는 `unshift`, 꼬리는 `pop`  
- 먹이를 먹으면 `pop` 생략 → 길이 증가  

### 🍏 Food 생성
- 격자 내 랜덤 위치 생성  
- 뱀 몸통과 겹치지 않도록 검사  

### ⚠️ 충돌 처리
- 벽 충돌 → 게임 오버  
- 자기 몸 충돌 → 게임 오버  

### ⌨️ 입력 처리
- 방향키로 이동 방향 변경  
- 반대 방향 즉시 회전 방지
